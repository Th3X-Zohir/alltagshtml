What is HTML?

HTML (Hypertext Markup Language) is the standard markup language used to create web pages. It defines the structure and content of a web page, including text, images, and multimedia.

"HTML is not a programming language, it is a markup language. It tells the browser how to display content on a web page."

HTML Elements

HTML elements are defined by tags. Here are some basic HTML tags:

  • <p> - defines a paragraph
  • <img> - defines an image
  • <a> - defines a hyperlink
  • <em> - defines emphasized text
  • <strong> - defines strongly emphasized text

HTML Attributes

HTML attributes provide additional information about HTML elements. Here are some basic attributes:

  • id - uniquely identifies an element
  • class - specifies one or more class names for an element
  • style - specifies inline CSS styles for an element
  • src - specifies the URL of an external resource, such as an image or script
  • href - specifies the URL of a hyperlink
